# ЛыткаринЪ — исследование рынка и стратегия запуска пивного бренда

Интерактивный одностраничный дашборд с маркетинговым исследованием и стратегией вывода на рынок собственного пивного бренда в среднем ценовом сегменте, дополненный детальными аналитическими отчётами.

## Возможности

- Интерактивный дашборд на одной странице с навигацией по разделам и плавным скроллом
- Графики на Chart.js: доли игроков рынка, динамика сегментов, карта позиционирования (bubble), радар сравнения конкурентов, структура себестоимости и маржинальности, рост категории сидра
- Анализ рынка пива РФ: размер, динамика, ключевые игроки и их доли
- Конкурентный анализ среднего ценового сегмента (Жигули Барное, Хамовники, Балтика, Сибирская Корона, Kozel и др.)
- Обзор подмосковных крафтовых конкурентов
- Gaps Analysis — поиск незанятых ниш
- Проработка продуктовой линейки (5 SKU на старте), экономики и маржинальности
- Разбор смежной категории (сидр), каналов дистрибуции и регуляторных требований
- Дорожная карта запуска и чек-лист готовности
- Два подробных текстовых отчёта в Markdown: общее исследование и глубокий анализ конкурентов

## Стек

- HTML / CSS (без сборки и фреймворков)
- JavaScript
- [Chart.js](https://www.chartjs.org/) (через CDN) — визуализация данных
- Google Fonts (Inter, DM Serif Display, Space Grotesk)

## Структура

```
index.html                              — интерактивный дашборд (вёрстка, стили, графики)
ЛЫТКАРИНЪ_ПИВО_ИССЛЕДОВАНИЕ.md          — исследование рынка и стратегия запуска
ЛЫТКАРИНЪ_АНАЛИЗ_КОНКУРЕНТОВ_ПИВО.md    — глубокий анализ конкурентов сегмента
```

## Запуск

Внешних зависимостей и шага сборки нет — Chart.js и шрифты подключаются через CDN.

Откройте `index.html` напрямую в браузере либо поднимите локальный сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```
