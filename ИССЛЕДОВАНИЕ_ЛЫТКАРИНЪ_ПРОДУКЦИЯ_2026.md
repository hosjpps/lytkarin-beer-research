# 1. РЕЗЮМЕ ДЛЯ РУКОВОДИТЕЛЯ

**Ситуация:** Завод ЛыткаринЪ (260+ рецептов, собственная лаборатория с ИТМО, экскурсии, действующие бренды Чиж & Co, Ослиная, Caspary Brau, ВДНХ, 13K Functional) находится в уникальной позиции для запуска двух направлений:

### Лимонады ФИЛЛС (от "feels" — чувствуется)
- **Рынок:** ~450-520 млрд руб., крафтовый сегмент растёт на 12-18% в год
- **Окно возможностей:** уход Coca-Cola/PepsiCo, рост интереса к локальным брендам
- **Дизайн:** градиентный минимализм (направление А) — пустая ниша в России
- **Стартовая линейка:** 6 SKU, банка 330 мл, ценник 90-130 руб.
- **Инвестиции в запуск:** ~3-5 млн руб.

### Пивной бренд ЛыткаринЪ
- **Рынок:** ~700-750 млрд руб., средний сегмент — основная зона роста
- **Окно возможностей:** национализация Балтики, продажа активов Heineken, сети ищут российских поставщиков
- **Позиционирование:** "Подмосковная мануфактура", ~100 руб. за банку 0.5л
- **Стартовая линейка:** 4-5 SKU (Светлое, Нефильтрованное, Пшеничное, Безалкогольное)
- **Инвестиции в запуск:** ~8-15 млн руб.

**Главное конкурентное преимущество обоих направлений:** реальная история, реальный завод, 260+ сваренных рецептов, лаборатория с ИТМО. Это нарратив, которого нет ни у кого.

# 2. РЫНОК ЛИМОНАДОВ РФ

## 2.1 Размер и динамика

| Показатель | Значение |
|---|---|
| Объём рынка газированных безалкогольных напитков (натуральный) | ~6.5-7.0 млрд литров |
| Объём в деньгах (розница) | ~450-520 млрд руб. |
| Доля крафтового/премиального сегмента | 5-8% в деньгах, 2-4% в литрах |
| Объём крафтового сегмента | ~25-40 млрд руб. |
| Рост крафтового сегмента | 12-18% в год |
| Прогноз общего роста 2025-2026 | 3-5% в год (денежный) |

## 2.2 Структурные изменения

- Coca-Cola и PepsiCo формально ушли, но заводы работают под новыми брендами (Добрый, Frustyle)
- Освободившаяся ниша заполняется российскими и турецкими/иранскими брендами
- Рост собственных торговых марок ритейлеров (ВкусВилл, Перекрёсток, Магнит)
- Потребительский патриотизм: "Сделано в России" стало конкурентным преимуществом
- Акциз на сахар с 1 июля 2023 (7 руб./л при содержании сахара >5 г/100 мл) стимулирует low-sugar рецептуры

## 2.3 Ценообразование (банка ~330 мл, розница)

| Сегмент | Цена | Примеры |
|---|---|---|
| Эконом / масс-маркет | 30-55 руб. | СТМ сетей, Добрый |
| Средний | 55-90 руб. | Черноголовка, Fantola, Laimon Fresh |
| Средний+ | 90-130 руб. | Ascania, Borjomi Flavored, ВкусВилл СТМ |
| Премиум | 130-200 руб. | Holy Water, Fever-Tree |
| Крафт / ультра-премиум | 200-400+ руб. | Bundaberg, локальный крафт |

**Рекомендация для ФИЛЛС:** ценовой сегмент "средний+" — **90-130 руб. за банку 330 мл.** Наибольший потенциал роста и маржинальность.

## 2.4 Ключевые тренды 2025-2026

### Натуральность и "чистый состав"
- Отказ от искусственных красителей и ароматизаторов
- Маркировка "без консервантов", "на натуральном сахаре" — ключевой триггер покупки

### Low sugar / Zero sugar
- 15-20% рынка уже занимают версии без сахара или с пониженным содержанием
- Стевия, эритритол как альтернативные подсластители
- Акциз на сахар стимулирует снижение

### Функциональные добавки
- Витамины (С, группы B), адаптогены (женьшень, элеутерококк)
- Пребиотики и пробиотики
- На старте — НЕ делать функциональным, добавить витамин C как бонус

### Необычные и "сложные" вкусы
- Ботанические: базилик, лаванда, розмарин
- Пряные: имбирь, куркума
- Региональные: облепиха, иван-чай

### Упаковка
- Минимализм для премиума, поп-арт для молодёжного
- Slim-банки (250 мл) набирают популярность
- Экологичная упаковка — тренд

# 4. ДИЗАЙН БАНОК

## 4.1 Вердикт: направление А (градиенты) — стратегически верный выбор

Из двух направлений:
- **Направление А** (в проекте): градиентные банки с чистым шрифтом — "Клубника Базилик" (розово-зелёный), "Виноград Крем-Сода" (фиолетово-бежевый)
- **Направление Б** (скрины): мазки краски, волнистые линии, горные пейзажи

**Рекомендация — однозначно Направление А.** Вот почему:

### Соответствие концепции
"ФИЛЛС" / "feels" / "чувствуется" — это про ощущения. Ощущения не имеют чётких границ. Градиент — визуальная метафора перетекания одного чувства в другое. Мазки краски и горы — это конкретные образы, а не абстрактные ощущения.

### Трендовость
Градиентный минимализм — тренд с длинным горизонтом (2020-2028). Мазки краски — пик уже пройден (2021-2023).

### Масштабируемость
Новый вкус = просто новая пара цветов. Для мазков/иллюстраций нужно разрабатывать новый визуал каждый раз.

### Пустая ниша в России
Ни одного сильного бренда с градиентным минимализмом на российском рынке лимонадов.

### Производство
Градиент стабильнее и проще в печати на алюминии.

## 4.2 Мировые ориентиры

| Бренд | Подход | Что взять |
|---|---|---|
| **Trip** (UK) | Пастельные градиенты, wellness-эстетика | Ближайший аналог ФИЛЛС |
| **Poppi** (US) | Яркие однотонные фоны, крупный лого | Shelf impact, поп-арт |
| **Olipop** (US) | Пастельные тона, ретро-типографика | Система линейки |
| **Liquid Death** (US) | Антидизайн, хэви-метал эстетика | Смелость отличаться |
| **Dash Water** (UK) | Ультраминимализм, белая банка | Премиальность через пустоту |
| **Recess** (US) | Мягкие градиенты, wellness | Цветовые переходы |

## 4.3 Формула банки ФИЛЛС

```
┌─────────────────────────┐
│                         │
│    ████ ГРАДИЕНТ ████   │  ← 70% поверхности:
│    ████ (цвет А) ████   │     двухцветный градиент
│                         │     (вертикальный)
│       Ф И Л Л С         │  ← Белый жирный гротеск
│                         │     ~40% ширины банки
│    КЛУБНИКА БАЗИЛИК     │  ← Тонкое начертание
│                         │     того же шрифта
│    ████ (цвет Б) ████   │
│    ████ ГРАДИЕНТ ████   │
│                         │
│         330ml           │  ← Минимальная инфо
└─────────────────────────┘
```

**Ключевые элементы:**
- Никаких иллюстраций фруктов, иконок, паттернов
- Лёгкий шум (grain/noise) на градиенте — тактильность и "живость"
- Матовый лак (soft-touch) по всей банке + глянцевый spot UV на логотипе
- Логотип на крышке банки (для выкладки сверху)

## 4.4 Цветовая палитра линейки

| Вкус | Градиент | Эмоция |
|---|---|---|
| Клубника-Базилик | Коралловый → мятно-зелёный | "чувствуется лето" |
| Виноград-Крем | Фиолетовый → тёплый бежевый | "чувствуется уют" |
| Манго-Маракуйя | Ярко-жёлтый → оранжевый | "чувствуется энергия" |
| Лимон-Бузина | Бледно-жёлтый → шампань | "чувствуется лёгкость" |
| Грейпфрут-Розмарин | Терракотовый → приглушённый розовый | "чувствуется вечер" |
| Личи-Роза | Нежно-розовый → жемчужный | "чувствуется нежность" |

**Эффект "радуги" на полке:** 6 банок рядом создают единый цветовой блок, различимый с 3-5 метров.

## 4.5 Типографика

- **Логотип "ФИЛЛС":** жирный геометрический гротеск, верхний регистр
- Слово короткое (5 букв), симметричное — хорошо работает крупно
- Буква "Ф" визуально сильная — может быть акцентом
- **Рекомендуемые шрифты (с качественной кириллицей):** TT Norms Pro (TypeType), Golos (Paratype), Stem (Paratype), Graphik (Commercial Type)
- **Размер:** логотип занимает не менее 40% ширины банки, читается с 1.5 метров

## 4.6 Shelf Impact

**Контекст российской полки:**
- Coca-Cola/Добрый — красный, зелёный (насыщенные)
- Черноголовка — пёстрые, ретро
- Fantola — кислотные цвета
- Энергетики — чёрный, неон

**Вывод:** пастельные градиенты ФИЛЛС будут резко выделяться. Как говорить тихо в шумной комнате — привлекает больше внимания, чем ещё один крик.

## 4.7 Что взять из направления Б (и только это)

- **Текстурный шум (grain)** — добавить на градиент для тактильности
- **1-2 тёмных варианта** (фиолетовый→чёрный, зелёный→чёрный) — как "вечерние/intense" версии в расширенной линейке

## 4.8 Чего НЕ делать

- Не смешивать направления А и Б в одной линейке
- Не добавлять фотографии фруктов (это масс-маркет)
- Не использовать больше 2-3 цветов на одной банке
- Не использовать латиницу в названии (кириллица = аутентичность в 2025-2026)

# 9. ДИЗАЙН И ПОЗИЦИОНИРОВАНИЕ ПИВА

## 9.1 Название: "ЛыткаринЪ" напрямую на банке

**Рекомендация:** использовать "ЛыткаринЪ" по модели "Хамовники" — без суббренда.

Формат: **"ЛыткаринЪ Светлое"**, **"ЛыткаринЪ Пшеничное"** и т.д.

**Почему:**
- Географическая привязка отлично работает в пиве (Хамовники, Жатецкий, Мюнхенское)
- Название уникально — нет ни одного конкурента
- Твёрдый знак "Ъ" = дореформенная Россия, традиция
- Малоизвестность города — это интрига, а не минус
- Экономия на создании и продвижении второго бренда

## 9.2 Целевая аудитория

- **Ядро:** мужчины 25-45, средний доход, города 100K+
- **Расширенная:** женщины 25-35 (через пшеничное/нефильтрованное/безалкогольное)
- **Психография:** ценят "цена-качество", читают этикетку, патриотичны в потреблении, "осознанный мейнстрим"

## 9.3 УТП (уникальное торговое предложение)

**"Подмосковная мануфактура. Пиво от людей, которые сварили 260 рецептов и выбрали лучшие."**

Это отстраивает:
- От корпораций (AB InBev, Балтика) — вы не конвейер
- От крафта (AF Brew, Jaws) — вы не хипстеры
- Вы — мануфактура. Что-то между, и это уникально.

## 9.4 Стартовая линейка пива (4-5 SKU)

| SKU | Стиль | ABV | Почему |

## 9.5 Дизайн банки пива

### Принципы
- **Минимализм с характером:** максимум 3 цвета, крупная типографика
- **Матовый финиш** (soft-touch) + выборочный глянец — как Kozel
- **Цветовое кодирование** по стилям (потребитель запоминает "зелёная = светлое")
- **Монограмма "ЛЪ"** крупно по центру, вордмарк ниже
- **Подстрочник** "крафтовая мануфактура" мелко

# 12. ОБЩАЯ СТРАТЕГИЯ ЗАВОДА

## 12.1 Архитектура брендов

```
                  ┌─────────────────┐
                  │   ЛЫТКАРИНЪ     │  ← Мастер-бренд (завод)
                  │  мануфактура    │     lytkarin.ru
                  └────────┬────────┘
                           │
            ┌──────────────┼──────────────┐
            │              │              │
     ┌──────┴──────┐ ┌────┴─────┐ ┌──────┴──────┐
     │   ФИЛЛС     │ │ЛыткаринЪ │ │  Другие     │
     │  (лимонады) │ │  (пиво)  │ │  бренды     │
     │  feels      │ │  банка   │ │  (будущее)  │
     └─────────────┘ └──────────┘ └─────────────┘
```

- **ФИЛЛС** — отдельный бренд со своей айдентикой (градиенты, "feels")
- **ЛыткаринЪ** — пиво под именем завода (как Хамовники у МПК)
- Общий нарратив: "всё с одного завода, 260+ рецептов, лаборатория ИТМО"

## 12.2 Синергии

1. **Общее производство** — завод уже варит пиво, добавить линейку лимонадов
2. **Общая дистрибуция** — одни и те же оптовики, одни и те же сети
3. **Перекрёстный маркетинг** — ФИЛЛС продвигает ЛыткаринЪ и наоборот
4. **Экскурсии** — "попробуй всё: и пиво, и лимонады" — усиление туризма
5. **HoReCa** — предлагать барам и ресторанам полный пакет (пиво + безалкогольное)

## 12.3 Что делать первым?

**Рекомендация: запускать ФИЛЛС первым.**

Почему:
- Меньше инвестиций (3-5 млн vs 8-15 млн)
- Нет рекламных ограничений (в отличие от алкоголя)
- Быстрее выход на маркетплейсы
- Создаёт узнаваемость бренда ЛыткаринЪ как производителя
- Тестирует каналы дистрибуции
- Привлекает внимание к заводу → проще заходить с пивом потом

**Пиво — через 3-4 месяца после запуска ФИЛЛС**, когда отлажены процессы маркировки, логистики, работы с сетями.
